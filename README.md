# Family Trees

Genealogical documentation for multiple families.

## Structure

Each family has its own folder under `/families`:

- **families/paliyath/** — Paliyath family tree and records

## Getting Started

1. Navigate to a family folder
2. Add genealogical data (names, dates, relationships, etc.)
3. Commit changes with descriptive messages

## Files and Organization

Each family folder may contain:
- `tree.md` — Family tree structure and relationships
- `records.md` — Biographical details and notes
- `images/` — Family photos and documents
- Other supporting files as needed

## Contributing

- Keep records accurate and up-to-date
- Use consistent formatting
- Include dates in YYYY-MM-DD format where applicable
- Reference sources for important information
